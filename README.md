# 小红书 AI 图文助手 Demo

这是一个纯静态前端 demo，覆盖以下核心链路：

- AI 生成图文：上传参考图、选择模版、补充 prompt、提交生成
- AI 生成队列：查看历史任务、状态、参考图，支持失败后重试
- 发布页：自动填充生成图片、标题、正文，允许手动二次编辑与补图
- AI 预审核：发布前调用审核链路，命中风险时弹出提示弹窗

## 本地运行

直接在当前目录启动一个静态服务即可：

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://localhost:4173
```

## 技术说明

- 不依赖后端，不需要构建工具
- 前端直接调用火山引擎 Ark 接口
- 图片生成默认调用 `doubao-seedream-4-0-250828`
- 文案生成与审核默认调用 `doubao-1-5-lite-32k-250115`
- 队列与草稿使用 `localStorage` 持久化

## 说明

- 如果浏览器环境下图片接口返回结构与当前解析逻辑不同，页面会自动回退到本地 SVG 占位图，保证 demo 可展示
- 页面里部分与 AI 无关的模块是 UI 占位，不包含真实业务逻辑
